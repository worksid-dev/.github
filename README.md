# .github Repository

このリポジトリは、Organization 全体で共通利用する **ガイドラインやテンプレート** を管理するためのものです。
ここに配置されたファイルは、Org 内の各リポジトリに自動的に適用されます。

## 📌 趣旨

- 行動規範、セキュリティポリシーなどの共通ドキュメントを一元管理
- Issue / Pull Request テンプレートの共通化
- Org 全体の開発ルールを明文化

## 🔀 ブランチ運用ルール

- **master / main ブランチ** および **develop / development ブランチ** への **直接 push は禁止** です。
- 変更は必ず **Pull Request 経由** で行い、レビューを経てマージしてください。
- その他のブランチ運用ルールは各プロジェクトのリポジトリ README を参照してください。

## 📂 含まれる主なファイル

- `CODE_OF_CONDUCT.md` : 行動規範
- `SECURITY.md` : セキュリティポリシー
- `ISSUE_TEMPLATE/*` : Issue テンプレート
- `PULL_REQUEST_TEMPLATE.md` : PR テンプレート
